# ScandalouS3-Clam

### Synopsis

```
Usage: scandalous3-clam.rb [options]

Options:
    -v, --verbose                    Verbose
    -d, --debug                      Debug
    -n, --dry-run                    Dry run
    -b, --bucket NAME                bucket name
    -m, --max-threads NUM            max threads (def. 5 + 2)
```
